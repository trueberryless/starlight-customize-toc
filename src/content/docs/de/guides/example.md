Anleitungen führen Nutzer durch eine bestimmte Aufgabe, die sie erledigen möchten, oft in einer Abfolge von Schritten.
Um eine gute Anleitung zu schreiben, musst du darüber nachdenken, was deine Nutzer erreichen wollen.
